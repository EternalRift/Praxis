# Praxis
Git repo for my Uni Course Work 

CamRTC:
Allows for streaming of a hosts local camera via WebRTC sockets to a client (Jeston Nano) for use in accelerated Image Processsing 

ScrnRTC: 
Allows for low latency screen sharing via WebRTC sockets to a given client (Headset Portion) to mitigate the use of long range HDMI/DP cables into the interface of the Praxis Headset
